# InstaMailChecker
Checks if the specified email is registered on Instagram.

## Requirements for C# version
```
dotnet6.0+
```

## Install and Run
```
git clone https://github.com/joaostack/InstaMailChecker && cd InstaMailChecker && dotnet run
```

## Bash Version
```
chmod +x Check.sh
```

## Run

```
./Check.sh
```
